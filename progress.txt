**Learnings:**
- COBE v2 removed onRender callback — use globe.update() in RAF loop instead
- COBE v2 has native arc support (from/to with arcColor, arcWidth, arcHeight)
- GSAP ScrollTrigger pin must be guarded on mobile (check window.innerWidth)
- Framer Motion whileInView + viewport: { once: true } is cleaner than IntersectionObserver for simple animations
- AnimatePresence mode="wait" for toggle animations (GameCapture, EnrichmentPipeline)
- useSyncExternalStore is the lint-clean way to read media queries in React
- next/dynamic with ssr:false is the lint-clean way to do hydration guards (vs useState+useEffect)
